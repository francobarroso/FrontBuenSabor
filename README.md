# Dashboard del Buen Sabor

## ¿Como usarlo?
_1.Clonar el repositorio._
_2.Abrir el proyecto y ejecutar 'npm install' en una terminal._
_3.Ejecutar el proyecto con el comando 'npm run dev'._
